Sistema operacional cliente: editor de textos e planilha eletrônica, apresentação eletrônica, internet e correio eletrônico Antes de conhecer um pouco os sistemas essenciais para o funcionamento dos computadores, é fundamental compreender as diferenças entre hardware e software.

Enquanto o hardware refere-se aos componentes físicos de um computador, o software engloba os programas e as instruções que dão vida a esses componentes, permitindo que realizem tarefas específicas.

Em resumo, o sistema operacional é um software que atua como intermediário entre o hardware de um computador e os programas de aplicação (Tanenbaum e Bos, 2016). O sistema operacional fornece uma interface conveniente e segura para que os programas de aplicação possam acessar e utilizar os recursos do sistema de forma eficiente. Os sistemas mais conhecidos são: Windows, Linux, Android e IOS.

Quando se refere a um sistema operacional, encontram-se versões do mesmo sistema para clientes e servidores. A versão servidor é projetada para computadores que desempenham esse papel específico. Nesses casos, o sistema operacional é configurado para lidar com hardware mais robusto, como múltiplos processadores, oferecendo recursos otimizados para suportar cargas de trabalho intensivas, em sua maioria alocados em datacenters. Já o sistema cliente é direcionado para máquinas utilizadas por usuários finais, que não requerem um hardware tão poderoso. Essas versões são projetadas para oferecer uma experiência de usuário eficiente e amigável, adequada para tarefas cotidianas e aplicativos de produtividade.

Dentro do sistema operacional, encontra-se uma variedade de softwares que desempenham funções essenciais para o funcionamento do computador e para a interação entre o usuário e o sistema. Alguns são nativos do sistema operacional, ou seja, já são habilitados na instalação do sistema operacional, como calculadora, calendário, bloco de notas, visualizador de imagens e vídeos, entre outros. Por outro lado, há softwares que não são padrões do sistema operacional, mas de grande importância nos universos acadêmico e empresarial. São ferramentas essenciais que aumentam a eficiência, a produtividade e a colaboração entre os membros da equipe.

Há ainda os softwares (aplicação) de navegação na internet, também conhecidos como navegadores web, que são aplicativos projetados para permitir que os usuários acessem e interajam com informações por meio do famoso “WWW” (world wide web). Editor de texto – digitar e salvar

Atualmente, encontra-se no mercado uma variedade de editores de texto. Eles podem ser aplicativos isolados, como o WordPad, por exemplo, ou embarcados em suítes de escritório, como Microsoft Office, Google Docs, LibreOffice e iWork, da Apple. Para conhecer mais sobre editores de texto, serão apresentadas as principais ferramentas do Microsoft Word. Grande parte dessas ferramentas você encontra na maioria dos editores de texto.

Digitando o texto

Ao abrir o editor de texto, o cursor estará posicionado na área de trabalho do editor. Esse será o ponto inicial do texto. Conforme você digita, o cursor segue para mostrar onde será inserido o novo caractere.

Salvando um documento

Para gravar o arquivo no computador, é preciso acessar as opções Salvar ou Salvar como. Ao salvar um documento, o editor de texto solicitará o nome que será dado ao arquivo.

Para salvar um documento do Microsoft Word, siga os passos:

  • Clique no botão Microsoft Office.
  • Clique no comando Salvar.
  • Digite um nome para o arquivo.
  • Clique no botão Salvar ou pressione a tecla Enter.

Você deve ter observado que, no menu, encontram-se as ferramentas Salvar e Salvar como. Qual é a diferença entre elas? O Salvar mantém o nome e o local onde foi salvo o documento. Já a opção Salvar como abre a janela para modificar o local e/ou o nome do documento, mas mantém o documento antigo salvo no seu local de origem. Porém, quando o documento é salvo pela primeira vez, não há diferenças entre essas duas opções.

Formatação de página

As margens da página consistem no espaço em branco à volta dos limites da página. Porém, é possível posicionar alguns itens nas margens, como cabeçalhos, rodapé, número de páginas, entre outros.

Na guia Layout da Página, no grupo Configurar Página, clique na opção Margens. Para margens com largura comum, clique em Normal. Ao clicar no tipo de margem pretendido, o documento será alterado totalmente e de modo automático para o tipo de margem selecionado.

Caso queira especificar definições de margens próprias, clique em Personalizar Margens e, em seguida, nas caixas Superior, Inferior, Esquerda e Direita, introduza os novos valores para as margens.

Editor de texto – correção ortográfica, gramatical e formatação de colunas

Correção ortográfica e gramatical

Os editores de texto apresentam ferramentas de verificação ortográfica e gramatical. Muitas vezes, quando se está digitando, aparece um sublinhado vermelho ou verde no texto. O sublinhado vermelho indica que existe um erro ortográfico, já o sublinhado verde indica que existe um erro gramatical. Para aplicar o verificador de ortografia e gramática do Microsoft Word, siga os passos:

  • Leve o mouse até a palavra sublinhada.
  • Clique com o botão direito do mouse.
  • Leia as sugestões do menu rápido (o Microsoft Word apresentará sugestões de palavras que se assemelham ao texto errado).
  • Clique na opção que corresponde à palavra desejada.

Configuração e formatação de colunas

Ao formatar um texto, é possível dividi-lo em colunas. A configuração de coluna serve para colocar um texto em duas ou mais colunas. Para realizar essa divisão no Microsoft Word, siga os passos:

  • Selecione o texto a ser dividido.
  • Acesse a guia Layout da Página no grupo Configuração da Página.
  • Clique sobre o ícone Colunas.

Inserindo imagens

Os editores de texto, além de ferramentas para edição de texto, também apresentam ferramentas para inserção de imagens. Para realizar essa inserção no Microsoft Word, siga os passos:

  • Clique na guia Inserir.
  • Clique no botão Imagem.
  • Escolha a imagem no seu computador.
  • Clique duas vezes sobre ela.
