Sistema operacional cliente: editor de textos e planilha eletrônica, apresentação eletrônica, internet e correio eletrônico

Antes de conhecer um pouco os sistemas essenciais para o funcionamento dos computadores, é fundamental compreender as diferenças entre hardware e software.

Enquanto o hardware refere-se aos componentes físicos de um computador, o software engloba os programas e as instruções que dão vida a esses componentes, permitindo que realizem tarefas específicas.

Em resumo, o sistema operacional é um software que atua como intermediário entre o hardware de um computador e os programas de aplicação (Tanenbaum e Bos, 2016). O sistema operacional fornece uma interface conveniente e segura para que os programas de aplicação possam acessar e utilizar os recursos do sistema de forma eficiente. Os sistemas mais conhecidos são: Windows, Linux, Android e IOS.

Quando se refere a um sistema operacional, encontram-se versões do mesmo sistema para clientes e servidores. A versão servidor é projetada para computadores que desempenham esse papel específico. Nesses casos, o sistema operacional é configurado para lidar com hardware mais robusto, como múltiplos processadores, oferecendo recursos otimizados para suportar cargas de trabalho intensivas, em sua maioria alocados em datacenters. Já o sistema cliente é direcionado para máquinas utilizadas por usuários finais, que não requerem um hardware tão poderoso. Essas versões são projetadas para oferecer uma experiência de usuário eficiente e amigável, adequada para tarefas cotidianas e aplicativos de produtividade.

Dentro do sistema operacional, encontra-se uma variedade de softwares que desempenham funções essenciais para o funcionamento do computador e para a interação entre o usuário e o sistema. Alguns são nativos do sistema operacional, ou seja, já são habilitados na instalação do sistema operacional, como calculadora, calendário, bloco de notas, visualizador de imagens e vídeos, entre outros. Por outro lado, há softwares que não são padrões do sistema operacional, mas de grande importância nos universos acadêmico e empresarial. São ferramentas essenciais que aumentam a eficiência, a produtividade e a colaboração entre os membros da equipe.

Word

Editor de texto – digitar e salvar

Atualmente, encontra-se no mercado uma variedade de editores de texto. Eles podem ser aplicativos isolados, como o WordPad, por exemplo, ou embarcados em suítes de escritório, como Microsoft Office, Google Docs, LibreOffice e iWork, da Apple. Para conhecer mais sobre editores de texto, serão apresentadas as principais ferramentas do Microsoft Word. Grande parte dessas ferramentas você encontra na maioria dos editores de texto.

Digitando o texto

Ao abrir o editor de texto, o cursor estará posicionado na área de trabalho do editor. Esse será o ponto inicial do texto. Conforme você digita, o cursor segue para mostrar onde será inserido o novo caractere.

Salvando um documento

Para gravar o arquivo no computador, é preciso acessar as opções Salvar ou Salvar como. Ao salvar um documento, o editor de texto solicitará o nome que será dado ao arquivo.

Para salvar um documento do Microsoft Word, siga os passos:

  • Clique no botão Microsoft Office.
  • Clique no comando Salvar.
  • Digite um nome para o arquivo.
  • Clique no botão Salvar ou pressione a tecla Enter.

Você deve ter observado que, no menu, encontram-se as ferramentas Salvar e Salvar como. Qual é a diferença entre elas? O Salvar mantém o nome e o local onde foi salvo o documento. Já a opção Salvar como abre a janela para modificar o local e/ou o nome do documento, mas mantém o documento antigo salvo no seu local de origem. Porém, quando o documento é salvo pela primeira vez, não há diferenças entre essas duas opções.

Formatação de página

As margens da página consistem no espaço em branco à volta dos limites da página. Porém, é possível posicionar alguns itens nas margens, como cabeçalhos, rodapé, número de páginas, entre outros.

Na guia Layout da Página, no grupo Configurar Página, clique na opção Margens. Para margens com largura comum, clique em Normal. Ao clicar no tipo de margem pretendido, o documento será alterado totalmente e de modo automático para o tipo de margem selecionado.

Caso queira especificar definições de margens próprias, clique em Personalizar Margens e, em seguida, nas caixas Superior, Inferior, Esquerda e Direita, introduza os novos valores para as margens.

Editor de texto – correção ortográfica, gramatical e formatação de colunas

Correção ortográfica e gramatical

Os editores de texto apresentam ferramentas de verificação ortográfica e gramatical. Muitas vezes, quando se está digitando, aparece um sublinhado vermelho ou verde no texto. O sublinhado vermelho indica que existe um erro ortográfico, já o sublinhado verde indica que existe um erro gramatical. Para aplicar o verificador de ortografia e gramática do Microsoft Word, siga os passos:

  • Leve o mouse até a palavra sublinhada.
  • Clique com o botão direito do mouse.
  • Leia as sugestões do menu rápido (o Microsoft Word apresentará sugestões de palavras que se assemelham ao texto errado).
  • Clique na opção que corresponde à palavra desejada.

Configuração e formatação de colunas

Ao formatar um texto, é possível dividi-lo em colunas. A configuração de coluna serve para colocar um texto em duas ou mais colunas. Para realizar essa divisão no Microsoft Word, siga os passos:

  • Selecione o texto a ser dividido.
  • Acesse a guia Layout da Página no grupo Configuração da Página.
  • Clique sobre o ícone Colunas.

Editor de texto – inserir imagens, tabelas e imprimir

Inserindo imagens

Os editores de texto, além de ferramentas para edição de texto, também apresentam ferramentas para inserção de imagens. Para realizar essa inserção no Microsoft Word, siga os passos:

  • Clique na guia Inserir.
  • Clique no botão Imagem.
  • Escolha a imagem no seu computador.
  • Clique duas vezes sobre ela.

Inserindo tabelas

Os editores de texto também apresentam ferramentas para edição de tabelas. Para realizar a inserção de tabelas no Microsoft Word, siga os passos:

  • Clique na guia Inserir, grupo Tabelas.
  • Clique na seta logo abaixo da palavra Tabela.
  • Escolha o tamanho da tabela (número de colunas e linhas).

Imprimindo

Após a digitação e a formatação, o texto está pronto para impressão. Antes de imprimir, é interessante visualizar a impressão. Dessa forma, você terá uma prévia de como o documento ficará após a impressão. Para visualizar a impressão e imprimir um documento de texto desenvolvido no Microsoft Word, siga os passos:

  • Clique no botão Microsoft Word.
  • Posicione o mouse na opção Imprimir.
  • Selecione a opção Visualizar Impressão e clique nela.

Para imprimir, clique na opção Sistema. Abrirá uma caixa de diálogo na qual você pode definir o que realmente deseja imprimir (todas páginas, intervalo de páginas, página atual ou somente seleção), o número de cópias, as páginas por folha, o tipo de folha etc. Após a configuração, clique no botão OK.

Excel – planilhas eletrônicas

Assim como os editores de texto, no mercado também há uma variedade de editores de planilhas eletrônicas. Esse software também compõe as suítes de escritório, como o Microsoft Office, com o Excel, e o LibreOffice, com o Calc. Para conhecer mais editores de planilhas, serão apresentadas as principais ferramentas do Microsoft Excel. Grande parte delas você encontra nos editores de planilhas eletrônicas.

Inserindo dados

Uma planilha é composta de várias células, e cada uma delas é a interseção de uma linha com uma coluna. É nas células que devem ser inseridos os dados da planilha. Elas aceitam dados como:

  • Textos
  • Números
  • Fórmulas
  • Hiperlinks
  • Funções
  • Data
  • Hora
  • Moeda
  • Contábil
  • Percentual
  • Personalizar

Para inserir um dado em uma célula, basta executar os seguintes passos:

  • Selecione a célula em que você deseja incluir informações clicando sobre ela com o mouse ou com as setas de direcionamento do teclado.
  • Digite a informação.

Selecionando células

Para realizar qualquer modificação, como cor, tipo de fonte e tamanho da fonte, é necessário selecionar a célula em que a rotina será aplicada. Para realizar uma seleção alternando células, linhas ou colunas inteiras, utilize a tecla Ctrl e clique sobre as células, as linhas ou as colunas que deseja selecionar.

Para realizar uma seleção em bloco, não alternando células, linhas ou colunas inteiras, utilize a tecla Shift e clique sobre as células, as linhas ou as colunas que deseja selecionar para formar o bloco.

Quebrando texto

Em alguns casos, quando se cria uma tabela, é necessário inserir informações que ocuparão mais de uma linha no cabeçalho dessa tabela. Neste caso, deve-se dividir o texto em duas linhas dentro da mesma célula.

Para aplicar uma quebra de texto em uma célula, siga estes passos:

  • Selecione a(s) célula(s).
  • Clique na guia Início.
  • Clique no ícone Quebrar texto automaticamente.

O mesmo acontece com as linhas, caso seja necessário aumentar ou diminuir sua altura.

Mesclando células

Em muitos casos, é preciso juntar duas ou mais células tanto na mesma coluna quanto na mesma linha. Essa rotina é muito utilizada para inserir títulos em planilhas.

Para mesclar células, siga os passos:

  • Selecione as células.
  • Clique na guia Início.
  • Clique no ícone Mesclar células.

Inserindo linhas e colunas

O Microsoft Excel possibilita a inserção de novas linhas ou de novas colunas nas planilhas. Para inserir novas linhas ou colunas, siga estes passos:

  • Clique com o botão direito no número da linha ou na letra da coluna.
  • Clique em Inserir e selecione a linha ou a coluna.

PowerPoint

Há uma variedade de editores de apresentações eletrônicas. Essa ferramenta vem embarcada nas suítes de escritório, como Microsoft Office e LibreOffice. Encontram-se editores de apresentações on-line como o Prezi, Canva e SlideShare, além do editor de slides disponível nas ferramentas Google. Para conhecer mais editores de apresentações eletrônicas, serão apresentadas as principais ferramentas do Microsoft PowerPoint.

A estrutura da apresentação é desenvolvida por meio de slides, ou seja, cada tela de apresentação no PowerPoint é chamada de slide.

Layout

Para cada slide, é possível definir um layout. Um exemplo são os slides de título, que vêm com opções para inserção de título e subtítulo (padrão do primeiro slide). Já na segunda tela, normalmente, utiliza-se o layout de texto, que vem com título e campo para digitar textos. Nesse formato de layout, há as seguintes opções:

  • Inserir tabela
  • Inserir gráfico
  • Inserir elemento do gráfico SmartArt
  • Inserir imagem
  • Inserir clip-art
  • Inserir clipe de mídia

Para mudar o layout do slide, siga os passos:

  • Selecione o slide.
  • Clique no botão Layout e escolha o modelo.

Design de slides

Na opção Design, selecionam-se os modelos disponíveis no editor de slides. Como padrão, a apresentação aparece em branco. Para definir um tema para apresentação, siga os passos:

  • Clique na guia Design.
  • Escolha um tema (para ver mais opções, clique na seta ao lado dos temas).

Transição de slides

É o efeito que se insere para suavizar a mudança entre uma tela e outra. Para inserir o efeito de transição em um slide, siga os passos:

  • Selecione o slide.
  • Clique na guia Animações.
  • Escolha a transição (ao clicar na transição, observe que ela acontece para você ver).

Apresentação de slides

  • Do Começo – Inicia a apresentação a partir do slide 1 (ou pressione F5 para iniciar a apresentação).
  • Do Slide Atual – Inicia a apresentação a partir do slide atual (o que está na tela).
  • Apresentação de Slides Personalizada – Você pode definir a ordem da apresentação.

Salvando a apresentação

A forma de salvar um documento no PowerPoint é semelhante à dos demais programas que compõem o Microsoft Office. Para salvar uma apresentação, siga a seguinte rotina:

  • Clique no botão do Office.
  • Clique na opção Salvar como.
  • Digite um nome para a apresentação.
  • Clique no botão Salvar.
E-mail, nuvem e seus recursos

E-mail

O conceito de enviar mensagens eletrônicas de texto entre partes de uma maneira análoga ao envio de cartas é anterior à criação da internet. Atualmente, há uma grande variedade de webmails, como Outlook, Gmail e iCloud, por exemplo. Além do envio de mensagens, os servidores de e-mail possibilitam o compartilhamento de arquivos, conhecidas como “nuvens”.

Utilizar a “nuvem” significa compartilhar ou hospedar algum arquivo por meio do browser, e, como esse arquivo está na web, diz-se que ele está no “ar”, na “nuvem”, ou seja, o arquivo não está fisicamente no desktop ou notebook.

Assim, há vários sistemas de armazenamento na web, ou “nuvens”, como o OneDrive, da Microsoft, o Google Drive, do Google, o WeTransfer, entre outros.

Veja o exemplo das rotinas de compartilhamento do Google por meio do Google Drive.

  • Acesse o link https://drive.google.com.
  • Digite seu usuário e sua senha do Gmail.
  • Clique em Fazer login.

O navegador abrirá a interface do Google Drive, como na figura:

Para compartilhar um arquivo no Google Drive, siga os passos:

  • Clique sobre o arquivo com o botão direito do mouse.
  • Clique em Compartilhar.
  • Clique no campo Adicionar pessoas.
  • Digite o e-mail (se for mais de um, separe-os por vírgula).

O usuário receberá uma mensagem no seu e-mail avisando que está sendo convidado a compartilhar um arquivo. Lembre-se de que o usuário que receberá o compartilhamento deve ter conta de e-mail no Google.

Canva

Funcionalidades do Canva

O primeiro passo é logar no site do Canva (canva.com) e fazer seu registro no lado superior direito. Você pode se cadastrar direto com a conta do Gmail ou do Facebook, por exemplo. Para logar no site, basta clicar direto na opção do seu cadastro, esperar carregar a página e pronto!

No menu principal do Canva, ao lado esquerdo, embaixo da foto, há a opção do Canva Pro, na qual você paga uma mensalidade para ter diversas opções extras para facilitar sua criação de conteúdo. Os valores variam com o tempo, mas é uma ótima opção para quem procura dar um upgrade em seus posts e vídeos com mais facilidade e com layouts mais modernos e criativos.

Voltando ao menu principal, há, ao lado esquerdo, modelos, layouts de diversos tamanhos e opções. A novidade é o Docs, uma alternativa de criação de documentos ao Google Docs. Além do texto em seus mais diversos formatos, ele permite inserir quase tudo dentro de um documento, como slides, gráficos, fotos e vídeos. Você pode usá-lo para criar documentos visuais em particular ou em colaboração com colegas.

Dentro do Canva, também estão os modelos predefinidos para Instagram, nos tamanhos certos. Você só precisa escolher o formato, post, storie ou reel, e editar com sua identidade visual, ou como queira.

Quando há um cifrão em cima do conteúdo, ao lado direito, então significa que esse conteúdo é pago. Você pode editar, mas não conseguirá fazer o download sem fazer o pagamento. Porém, é possível utilizar outros modelos. Por exemplo, em uma tirinha, mesmo estando em preto e branco, é possível fazer ajustes como cor de fundo, do personagem, das legendas, ou até mesmo trocar o layout e inserir mais personagens, se quiser.

O Canva salva automaticamente suas edições, mas, para não deixar seus trabalhos desorganizados, você pode criar uma pasta clicando em Arquivo, Salvar em pasta, Seus projetos e Criar nova. Para acessá-la outras vezes, clique na opção Ir para a pasta ou vá até o menu principal, no lado esquerdo, clique em Projetos e selecione a aba Pastas.

Uma opção é deixar o site do Canva salvo na barra de favoritos – é mais prático e ágil. Basta você clicar na estrela ao lado direito da barra da página do navegador e clicar em Concluir.

O Canva também é uma ótima opção para quem gosta ou precisa trabalhar pelo celular. O aplicativo é muito prático e encontra-se em qualquer loja de aplicativo para baixar de forma gratuita. Basta pesquisar o nome “Canva”. Para o login, é o mesmo processo do site, basta registar seu e-mail e sua senha.

Dicas para digitação

Quem utiliza o computador para trabalhar, estudar e até mesmo para acessar redes sociais sabe como algumas funcionalidades são fundamentais e melhoram a experiência de uso. Conhecer bem o teclado e suas diversas funções pode aumentar significativamente a produtividade, economizando tempo e tornando as tarefas diárias mais eficientes.

Os atalhos de teclado, por exemplo, são uma maneira rápida de executar comandos sem precisar navegar pelos menus. Eles podem variar entre sistemas operacionais, mas muitos são universais e funcionam em diversos aplicativos e plataformas.

Veja, a seguir, algumas dicas para você testar, mas lembre-se de que essas funcionalidades são compatíveis com teclados de computadores que utilizam o sistema operacional Windows.

Dicas de digitação:

Funcionalidade Atalho
Colocar primeira letra maiúscula Pressione a tecla Shift e a letra desejada (em alguns teclados o Shift é uma seta).
Colocar várias letras maiúsculas Pressione a tecla Caps Lock e digite a palavra.
Dar espaço entre palavras Pressione a barra de espaço.
Acentuar (ã, õ, á, é, ó) Pressione a tecla com acento e depois a letra que deseja acentuar.
Acentuar (â, ê, ô, à) Pressionando a tecla Shift, clique na tecla com acento e depois na letra.
Trocar de linha Pressione a tecla Enter.
Obs.: um parágrafo pode ter cinco linhas, por isso pressione a tecla Enter somente quando ele terminar.
Apagar texto Para apagar um texto à esquerda do cursor, clique na tecla Backspace (fica acima da tecla Enter).
Para apagar um texto à direita do cursor, clique em Delete (ou Del).
